# WebdriverIO-example
Basic script using webdriverIO and Appium

### Prerequisites 
* NodeJS 8+
* Npm

### Installing

A step by step series of examples that tell you how to get a development env running

* Clone the project
* Run: `npm install`

## Running the tests
* Run: `npm test` << To Run the tests.
* The results are handled by `spec reporter`
